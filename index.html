How to play?

  • When you start, first select the point where you want to change the color, then it will be selected, you can set the color of it with the color selector on the top!
  • If you have colored the circles you can continue by clicking the "Next Row" button, it will lock the previous row and will unlock the next one!
  • Also will it show colored dots on the side of the row, each gray dot means theres a wrong color, each white dot means theres a color that the answer includes, but not on the right position, each red dot means theres a color on the right spot and is in the answer!
  • If you lose or win, it will show on the top!
  • WARNING: there can be multiple of the same color in the answer!
